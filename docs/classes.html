Utility Classes for
your markup .

Tip:

In order to quickly become familiar with this utility class framework you must become familiar with 1) eHealth Design Specs and 2) Function prefixes. Design specifications are fed directly into CSS class generating functions to provide the classes to always deliver features on brand.

The main specs defined are for:

  • Colors (background-color, border-color, and text-color)
  • Spacing (margin, padding, max-width, and height)
  • Typography

Quick Naming Cheat Sheet

Color PREFIXES

  • .bg-* (background-color)
  • .border-* (border-color)
  • .text-* (color | font)

Color NAMES

  • black
  • white
  • grey-dark
  • grey
  • grey-light
  • primary
  • primary-light
  • primary-lightest
  • func-primary
  • func-secondary
  • func-secondary-light

Spacing PREFIXES

  • .m-* , .mx-* , my-* , m(first letter)-* (margin)
  • .p-* , .px-* , py-* , p(first letter)-* (padding)
  • .max-w-* (max-width)
  • .h-* (height)

Spacing Names

  • xxxxs
  • xxxs
  • xxs
  • xs
  • sm
  • base
  • md
  • lg
  • xl
  • xxl
  • xxxl
  • xxxxl
  • component

Borders (ex. border: 4px solid #000)

  • .border-* (widths: 0, 1, 2, 4, 8)
  • .border-* (styles: solid, dashed, dotted, none)
  • .border-* (colors: see Color Names)

Typography

  • .headline
  • .h1
  • .h2
  • .h3
